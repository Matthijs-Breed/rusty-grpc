 # Project setup
 - Local code in ./src
 - Classes in Voting and GPBMetadata are autogenerated based on ../protos/voting.proto
 - ./src classes manually added to autoload via composer.json:
 ```
    ...
     "autoload": {
        "psr-4": {
            "Voting\\": "src/Voting/",
            "GPBMetadata\\": "src/GPBMetadata/"
        }
    },
    ...
```
